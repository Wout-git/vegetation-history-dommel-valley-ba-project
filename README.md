# Bachelor's Project: Vegetation History of the Dommel Valley

This repository contains the bachelor thesis and analyses for a paleoecological study on the vegetation dynamics of the Dommel Valley conducted at the University of Amsterdam, focused on pollen analysis (microscopically identified by myself) and multiproxy reconstruction, including charcoal and phytolith analyses conducted by other students.

## Files

- `bechelor_thesis_dommel_valley_vegetation_history.pdf` — Bachelor thesis manuscript
- `wolfwinkel2_pollen_analysis.R` — R script for pollen and paleoecological analysis

## Focus

- Holocene vegetation dynamics
- Pollen-based reconstruction
- Dommel Valley paleoecology
- Human and fluvial landscape influence

## Data Collection

The data were obtained from the Wolfwinkel 2 sediment core collected in February 2025 from an abandoned channel of the Dommel River in the Netherlands.

## Data Analysis

The analysis includes pollen zonation (CONISS), DCA ordination, linear regression, and regional comparison with pollen records retrieved from the Neotoma Paleoecology Database using R for chronostratigraphic interpretation and paleoecological reconstruction.
